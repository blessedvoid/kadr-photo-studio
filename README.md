# kadr-photo-studio
📸 Фотостудия «Кадр»

Веб-сайт фотостудии «Кадр» — современный и удобный интерфейс для просмотра каталога фотосессий, онлайн-записи и получения контактной информации.

🚀 Функциональность
Каталог фотосессий (портретная, семейная, love story и др.)
Фильтрация и сортировка услуг
Онлайн-запись на фотосессию
Контактная информация и форма обратной связи
Адаптивный дизайн (десктоп и мобильные устройства)

🛠️ Технологии
HTML5
CSS3
JavaScript (Vanilla)

📁 Структура проекта
/css      — стили
/js       — скрипты
/images   — изображения
/fonts    — шрифты (если используются локально)
index.html — главная страница

⚙️ Запуск проекта
Склонируйте репозиторий:
git clone https://github.com/your-username/kadr-photo-studio.git
Откройте файл index.html в браузере.
